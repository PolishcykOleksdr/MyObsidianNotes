1. Спілкуємося з ChatGPT як професіонали — міні-посібник (до 500 слів):
```
Поясни, як правильно користуватися ChatGPT.
Напиши короткий посібник для початківців (до 500 слів), у якому розкажи:
– що таке промти,
– що таке праймінг,
– як створювати корисних персонажів.
Наведи по 2–3 приклади для кожного поняття.
Дай відповідь українською мовою.
```

2. Опановуємо будь-яку навичку за 30 днів — чіткий план від нейромережі:
```
Я хочу вивчити [назва навички].
Склади покроковий 30-денний план навчання для повного новачка.
Включи щоденні завдання, контрольні точки та формати навчання (текст/відео/практика).
Дай відповідь українською мовою.
```

3. Пояснення будь-якої складної теми простою мовою:
```
Поясни мені [складна тема], ніби мені 12 років.
Використовуй аналогії, просту мову та реальний приклад із життя.
Наприкінці задай 3 питання, щоб перевірити, наскільки я зрозумів(ла) тему.
Дай відповідь українською мовою.
```

4. Навчання за принципом 80/20 — тільки суть, без зайвої води:
```
Я хочу вивчити [встав тему].
Визнач 20% концепцій або навичок, які дають 80% результату на практиці.
Склади їх у вигляді пріоритетного списку з короткими поясненнями та прикладами застосування.
Дай відповідь українською мовою.
```

5. Перевірка отриманих знань із зворотним зв'язком:
Зараз я вивчаю [встав тему].
```
Перевір мої знання — задай мені 5–7 питань, які стають дедалі складнішими.
Після кожної моєї відповіді дай зворотний зв'язок і поясни правильну відповідь простими словами.
Дай відповідь українською мовою.
```
