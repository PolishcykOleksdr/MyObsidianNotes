>**Як отримати число в потрібному діапазоні?** Стандартний інструмент `Math.random()` видає дробове число від **0.0** до **1.0**. Щоб перетворити цей "шматочок" на ціле число (наприклад, для гри в кості від 1 до 6), ми використовуємо математичне масштабування та зміщення.

---

> [!abstract] Core Framework
> 
> **Генерація діапазону** — це процес розтягування стандартного інтервалу $[0, 1)$ до потрібної ширини (`max - min`) та його перенесення у потрібну точку (`+ min`).
> 
> **Візуальна метафора:** Це як **гумова стрічка**. Спочатку вона має довжину 1 см (`Math.random`). Ви розтягуєте її до потрібної довжини діапазону, а потім переносите (кладете) її на лінійці в те місце, де починається ваше мінімальне число.

---

> [!info] Key Insights
> 
> - **Math.random():** Завжди повертає `double` у напіввідкритому інтервалі $[0.0, 1.0)$. Це означає, що 0.0 може випасти, а 1.0 — ніколи.
>     
> - **Множник (max - min + 1):** Визначає кількість можливих варіантів (ширину вікна). Додавання `+1` необхідне, щоб включити верхню межу (`max`) у результат.
>     
> - **Приведення типів (int):** Відкидає дробову частину. Це завжди округлення в меншу сторону.
>     
> - **Зсув (+ min):** Гарантує, що результат не буде меншим за встановлений мінімум.
>     

---

> [!example] Класична формула
> 
> $$random = (int) (Math.random() \times (max - min + 1)) + min$$
> 
> **Приклад для діапазону від 20 до 59:**
> 
> 1. Ширина: $59 - 20 + 1 = 40$ можливих значень.
>     
> 2. Множимо: `Math.random() * 40` дає результат від `0.0` до `39.999...`.
>     
> 3. Перетворюємо в `int`: маємо цілі числа від `0` до `39`.
>     
> 4. Додаємо `min`: $0 + 20 = 20$, а $39 + 20 = 59$.
>     

---

> [!abstract] Альтернатива: Клас java.util.Random
> 
> Якщо вам потрібно генерувати багато чисел, зручніше використовувати спеціальний клас `Random`. Він пропонує готові методи, які роблять код чистішим.
> 
> Java
> 
> ```java
> import java.util.Random;
>
> Random random = new Random();
> // Генерує число від 0 (включно) до 100 (не включно)
> 
> int simple = random.nextInt(100);
> // Від 1 до 100 (не включно)
> 
> int range = random.nextInt(min, max + 1);
> // Сучасний спосіб для діапазону [min, max] (Java 8+)
> ```
