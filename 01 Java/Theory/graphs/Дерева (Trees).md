Дерево — це абстрактна структура даних, що складається з вузлів, з'єднаних між собою гілками. Вони використовуються для моделювання ієрархічних структур: від файлових систем до DOM-моделей XML та HTML.

---

> [!abstract] Core Framework
> 
> Дерево моделює ієрархію, де кожен елемент (крім кореня) має одного "батька" і нуль або більше "дітей".
> 
> - **Колекції Java:** Вбудовані структури (`TreeMap`, `TreeSet`) базуються на збалансованих червоно-чорних деревах.
>     
> - **Зв'язки:** Кожна **Гілка** (Edge) вказує напрямок від батьківського вузла до дочірнього.
>     
> - **Метрики:** **Рівень** (Level) показує відстань від кореня, а **Висота** (Height) — максимальну кількість рівнів.
>     

---

> [!info] Key Insights
> 
> - **Коренева логіка.** Усе дерево починається з одного вузла — **Коріння** (Root).
>     
> - **Кінцеві точки.** Вузли без дочірніх елементів називаються **Листами** (Leaf).
>     
> - **Піддерева.** Будь-яка частина дерева, що виходить з одного вузла, сама є повноцінним деревом.
>     
> - **Ефективність.** Збалансовані бінарні дерева гарантують виконання операцій за логарифмічний час $O(\log n)$.
>

### Основні типи бінарних дерев

1. **Бінарне дерево (Binary Tree):** Кожен вузол має максимум два дочірніх вузли.![[binaryTree.png]]

2. **Троїчне дерево (Ternary Tree):** Кожен вузол може мати до трьох дочірніх вузлів.![[ternaryTree.png]]
3. **Ідеальне бінарне дерево (Perfect Binary Tree):** Усі рівні повністю заповнені.![[perfectBinaryTree.png]]
4. **Завершене бінарне дерево (Complete Binary Tree):** Заповнене зліва направо без пропусків. ![[Complete Binary Tree.png]]
5. **Повне бінарне дерево (Full Binary Tree):** Кожен вузол має або 0, або 2 нащадки. ![[Full Binary Tree.png]]
6. **Збалансоване бінарне дерево (Balanced Binary Tree):** Різниця висот піддерев не більше 1. ![[Balanced Binary Tree.png]]
# Приклад коду: Ручне створення

```java
/* Клас для представлення вузла дерева */
class TreeNode {
    int data;
    TreeNode left;
    TreeNode right;

    public TreeNode(int data) {
        this.data = data;
        left = null;
        right = null;
    }
}

public class ManualTree {
    public static void main(String[] args) {
        /* Створення ієрархії вручну */
        TreeNode root = new TreeNode(1);
        root.left = new TreeNode(2);
        root.right = new TreeNode(3);
        
        /* Додавання листя */
        root.left.left = new TreeNode(4);
        root.left.right = new TreeNode(5);
    }
}
```

---

> [!warning] Обмеження та ризики
> 
> - **Виродження.** Без балансування дерево може перетворитися на лінійний список, втрачаючи переваги швидкості пошуку.
>     
> - **Витрати пам'яті.** Кожне посилання (гілка) займає місце в пам'яті JVM.
>     
> - **Складність видалення.** Видалення вузла (особливо кореня або внутрішнього вузла) вимагає переналаштування зв'язків для збереження структури.
>
