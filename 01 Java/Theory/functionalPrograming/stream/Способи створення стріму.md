# Є 6 способів як можна створити stream відповідно до різних ситуацій

>[!info] З колекції
>Стрім можна легко створити з існуючої колекції, використовуючи метод stream().
# Наприклад:

```java
import java.util.Arrays;  
import java.util.List;  
import java.util.stream.Stream;
public class StreamCreationFromCollection {
    public static void main(String[] args) {
	    List<String> fruits = Arrays.asList("apple", "orange", "banana", "grape");
	    
        // Створення стріму з колекції
        Stream<String> fruitStream = fruits.stream();
        
        // Використання стріму
        fruitStream.forEach(System.out::println);
    }
}
```

---

>[!info] Зі значень
>Використовуючи метод Stream.of(), можна швидко створити стрім із заданих значень.
# Наприклад:

```java
import java.util.stream.Stream;

public class StreamCreationFromValues {
    public static void main(String[] args) {
        // Створення стріму зі значень
        Stream<String> fruitStream = Stream.of("apple", "orange", "banana", "grape");

        // Використання стріму
        fruitStream.forEach(System.out::println);
    }
}
```
---

>[!info] Зі згенерованих значень
>Метод Stream.generate() дозволяє створювати стрім із згенерованих значень. Наприклад, створимо стрім із п'яти випадкових чисел.
# Наприклад:

```java
import java.util.stream.Stream;

public class StreamCreationGeneratedValues {
    public static void main(String[] args) {
        // Створення стріму зі згенерованих значень
        Stream<Integer> numberStream = Stream.generate(() -> (int) (Math.random() * 100));

        // Використання стріму
        numberStream.limit(5).forEach(System.out::println);
    }
}
```

---

>[!info] З ітератора
>Якщо у вас є ітератор, ви можете використовувати метод Stream.iterate() для створення стріму.
# Наприклад:

```java
import java.util.Arrays;
import java.util.Iterator;
import java.util.stream.Stream;

public class StreamCreationFromIterator {
    public static void main(String[] args) {
        Iterable<String> iterableFruits = Arrays.asList("apple", "orange", "banana", "grape");
        Iterator<String> fruitIterator = iterableFruits.iterator();

        // Створення стріму з ітератора
        Stream<String> fruitStream = Stream.iterate(fruitIterator.next(), f -> fruitIterator.hasNext(), f -> fruitIterator.next());

        // Використання стріму
        fruitStream.forEach(System.out::println);
    }
}
```

---

>[!info] З рядка
>Розбивши рядок на слова, можна отримати стрім слів за допомогою методу Stream.of().
# Наприклад:

```java
import java.util.stream.Stream;

public class StreamCreationFromString {
    public static void main(String[] args) {
        String str = "This is a sample string";

        // Створення стріму з рядка
        Stream<String> wordStream = Stream.of(str.split("\\s+"));

        // Використання стріму
        wordStream.forEach(System.out::println);
    }
}
```

---

>[!info] Порожній стрім
>Інколи вам може знадобитися порожній стрім, який можна створити за допомогою методу Stream.empty().
# Наприклад:

```java
import java.util.stream.Stream;

public class EmptyStreamCreation {
    public static void main(String[] args) {
        // Створення порожнього стріму
        Stream<String> emptyStream = Stream.empty();

        // Використання стріму
        emptyStream.forEach(System.out::println);
    }
}
```

### Ці методи надають різноманітні способи створення стрімів в Java, враховуючи різні джерела даних. Обирайте той, який найкраще відповідає вашому конкретному випадку використання, щоб зробити ваш код більш зрозумілим та ефективним.
