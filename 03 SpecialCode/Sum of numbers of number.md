```java
public class DRoot {  
    public static void main(String[] args) {  
        System.out.println(digital_root(56));  
    }  
    public static int digital_root(int n) {  
        return (n != 0 && n%9 == 0) ? 9 : n % 9;  
    }  
}
```

> [!warning] **Математична логіка** 
>  Приклад для числа **456**
> 1. Беремо цифри числа: **4, 5, 6**.
> 2. Додаємо їх: $4 + 5 + 6 = 15$.
> 3. Оскільки 15 — це ще не одна цифра, додаємо знову: $1 + 5 = 6$.
> 4. **Результат: 6.**

> [!question] Як працює цей Java-код (Магія числа 9)
> 
> Замість того, щоб змушувати комп'ютер реально розбивати число на цифри і додавати їх у циклі, програмісти використовують властивість числа **9**.
> 
> **Цікавий факт:** Залишок від ділення будь-якого числа на 9 буде дорівнювати сумі його цифр (або цифровому кореню).
> 
> Розберемо рядок:
> 
> `return (n != 0 && n % 9 == 0) ? 9 : n % 9;`
> 
> Це тернарний оператор (коротка версія `if-else`), який працює так:
> 
> 1. **Якщо число ділиться на 9 без остачі (і воно не нуль):**
>     
>     Наприклад, для числа **18**: 1 + 8 = 9. Але 18 % 9 дасть $0$. Оскільки ми хочемо отримати цифру 9, а не 0, код каже: "Якщо остача 0 — поверни 9".
>     
> 2. **В усіх інших випадках:**
>     
>     Просто поверни остачу від ділення на 9 (`n % 9`).

### Приклад по ітераціях для числа 56

Як код обробить число `56` з твого прикладу:

|**Крок**|**Дія в коді**|**Логіка**|**Результат**|
|---|---|---|---|
|**1**|Вхідне число `n = 56`|Перевіряємо умови.|$n = 56$|
|**2**|`n != 0`|Чи 56 не дорівнює 0?|Так (True)|
|**3**|`n % 9 == 0`|$56 / 9 = 6$ (остача **2**). Чи дорівнює остача 0?|Ні (False)|
|**4**|Результат `n % 9`|Оскільки умова не справдилася, беремо остачу.|**2**|
**Перевірка "вручну":** $56 \rightarrow 5 + 6 = 11 \rightarrow 1 + 1 = \mathbf{2}$.
